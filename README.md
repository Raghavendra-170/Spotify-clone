## Spotify-clone
This project is a front-end clone of the Spotify web player interface. Built using HTML and CSS, it replicates the layout and design of Spotify's user interface for desktop and mobile views.

## 📁 Project Structure

├── index.html # Main HTML file containing the structure of the app
├── style.css # CSS file for styling the interface
├── assets/ # Folder for icons, album covers, etc.

## 🚀 Features

- Responsive layout (desktop and mobile)
- Sidebar with navigation and library section
- Main content area with featured playlists and charts
- Sticky header with navigation options
- Music player UI with playback controls and song info

## 🎨 Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- Font Awesome (for icons)
- Google Fonts (Montserrat and Poppins)

## 📦 How to Use

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Explore the layout and music player styling.

## 🔧 Customization

You can customize:
- The album images and song details in the cards.
- The music player with JavaScript functionality (optional enhancement).
- Theme colors and responsiveness further as needed.
